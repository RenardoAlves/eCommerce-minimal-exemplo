Passo a passo para fazer o site funcionar localmente:

OBS: Front-end e Back-end devem estar rodando simultaneamente, cada um em seu terminal.

Front-end: \
1 - Entre na pasta /front-end \
2 - Execute o comando "npm install" \
3 - Crie um arquivo .env com uma linha: "VITE_API_URL=http://localhost:3000" \
4 - Execute o comando "npm run dev"

Back-end: \
1 - Entre na pasta /back-end \
2 - Execute o comando "npm install" \
3 - Execute o comando "npm start"
